# [baconbao.github.io](https://baconbao.github.io)


### Build

```
npm run build
```

The outputs will be placed in `dist` folder.

### Launch server

```
npm run serve
```

The site can be browsered in `localhost:3000`.

### Deploy

```
npm run deploy
```

[Warning!] Site will be built and pushed into remote `gh-pages` branch.
